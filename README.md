# AI-Engineering-Final-Project
This file is a Streamlit web app. You must start it with the streamlit command from a terminal.
1. Make sure dependencies are installed
In VS Code, open a terminal (Ctrl+`) and run (in the same environment your Python interpreter uses):
pip install streamlit scikit-learn pandas numpy requests rapidfuzz beautifulsoup4 lxml
If you’re using conda/virtualenv, activate it first.
2. Run the app correctly from VS Code
In the same terminal inside VS Code, run:
streamlit run DRUAUTH.py
You should see something like:
  You can now view your Streamlit app in your browser.

  Local URL: http://localhost:8501
Then:
Your browser may open automatically.
If not, open a browser and go to: http://localhost:8501.
That’s where the UI runs.
